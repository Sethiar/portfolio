Japaninfrane

Type

Site de vente en ligne / e-commerce

Objectif

Centraliser la vente de figurines japonaises via la collaboration avec des sites existants, en récupérant un pourcentage sur chaque vente grâce à l’intégration de leurs stocks.

Technologies

Frontend : Vue.js (desktop & mobile) ; Backend : Django + Django REST Framework (API RESTful, JWT Authentication); Intégration API Strike pour paiement ; API des partenaires pour récupération des stocks

Base de données

PostgreSQL

Hébergement souhaité

Docker / VPS (backend + frontend + base de données)

Fonctionnalités clés

  • Authentification utilisateurs et administrateur
  • Système de panier avec gestion des stocks via API partenaires
  • Intégration API de paiement Strike pour transactions sécurisées
  • Système de fidélisation avec points cumulables et suivi
  • Responsive design mobile et desktop
  • Interface administrateur : gestion des produits, commandes, utilisateurs et statistiques

Défis techniques / Ce que j’ai appris

  • Intégration simultanée de plusieurs APIs partenaires pour récupérer les stocks en temps réel
  • Gestion des transactions et sécurité avec API de paiement
  • Architecture multi-plateforme (mobile & desktop) avec React / Vue.js
  • Optimisation des performances pour synchronisation de stock et affichage des produits
  • Implémentation d’un système de fidélisation et suivi des points utilisateurs
